# singbox-lean

极简 sing-box 容器，不同构建 tag 选择情况下，可以小于 20MB。

注意：没有 nftables，所以不能使用基于 nftables 的 auto_redirect 功能。

**用法：**

Fork 本仓库，然后 Actions -- custom sing-box build -- Run workflow，按需求填写构建参数。
